# drush-module-cleanup

## Installation

* Put the nexteuropa_export_data.drush.inc file into your $HOME/.drush/ folder
* Run drush cc drush
```
cp nexteuropa_export_data.drush.inc ~/.drush/
drush cc drush
```
That's it!

## Usage

```
drush ne-export-data
drush ne-export-data --result-file=./result.csv
```
