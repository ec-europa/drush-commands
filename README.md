# drush-module-cleanup

## Installation

* Put the nexteuropa_cck_export.drush.inc file into your $HOME/.drush/ folder
* Run drush cc drush
```
cp nexteuropa_cck_export.drush.inc ~/.drush/
drush cc drush
```
That's it!

## Usage

```
drush ne-cck-export
drush ne-cck-export --result-file=./result.csv
```
